# pandas-csv-analysis
A Python Pandas project demonstrating how to load CSV files and perform basic data exploration using functions like read_csv(), head(), tail(), info(), and describe().

# Pandas CSV Data Analysis

This project demonstrates how to load and explore CSV datasets using Python Pandas.

Functions used:
- read_csv()
- head()
- tail()
- info()
- describe()

Datasets:
1. covid19.csv – Sample COVID case data
2. employee.csv – Employee dataset

Purpose:
To practice basic data exploration using Pandas.
